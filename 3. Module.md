# Module 是什么？

* 大道至简，Module其实就是公共函数，你写`Python`以及语言的时候，都会把公共功能封装成函数，然后可以复用。太阳底下没有新鲜事，Terraform也是一样。

* 比如我们要创建100个`EC2`，如果一个个去编写，会累死，而且冗余太多。这种情况下，我们就可以把它做成一个Module，然后传递变量即可。

* 实际上Terraform registry已经有很多优秀Module了，我们只需要调用即可,module可以在github或者本地，注意权限和路径即可。

* 例如： 
  
  ```
  module "ec2_instance" {
  source = "terraform-aws-modules/ec2-instance/aws" # 直接用官方的Module

  for_each = local.env[terraform.workspace].instances
 
  name = each.key

  ami                         = each.value.ami
  instance_type               = each.value.instance_type
  key_name                    = try(each.value.key_name, null)
  iam_instance_profile        = try(each.value.iam_instance_profile, null)
  vpc_security_group_ids      = [module.security_group.security_group_id]

  user_data_base64            = base64encode(each.value.user_data)
  root_block_device           = try(each.value.root_block_device, null)
  tags = {
    Terraform   = "true"
    Environment = "production"
    role        = each.value.role
  }
}

```
