# 版本差异


可以指定TF和provider的最低版本，比如：
```
terraform {

  required_version = ">= 1.7.0"

  required_providers {
    aws = {
    
      version = ">= ..."
    }
  }
}

```
# 维护多个TF版本的代码，用`asdf`比较理想

```
1. 安装，参考官方，略过。

2. 设置环境变量。~/.bash_profile
export PATH="${ASDF_DATA_DIR:-$HOME/.asdf}/shims:$PATH"

3. 安装plugin, 它有很多Plugin

asdf plugin list all 可以看到它的plugin
asdf plugin add terraform https://github.com/asdf-community/asdf-hashicorp.git

4. 安装 tf 的某个版本
asdf install terraform xxx 

5.进入代码目录，执行
asdf set terraform 0.14.3
该目录下的.tool-versions 会记录tf的版本，注意如果该版本不存在，则返回 4安装即可。
```
