
 # Backend 
 
* 之前说过，terraform state文件默认在本地，没办法共享以及协作，官方提出了`backend`的解决方案，思路就是把文件放在云端以及其他可以共享的地方。

* state文件的并发编辑保护用 `dynamoDB`, 它会阻止并发操作，确保文件的一致性。

* 官方支持的Backend非常多，既然你用到云了，建议用S3对象存储，自带了version管理和加密功能。 

* 如果你有3个环境，需要维护3套变量，workspace就是为这个准备的。

```
terraform workspace new dev 
terraform workspace list 
```
可以看到你的workspace，他内置了一个环境变量`env[terraform.workspace]`,这个返回值就是你的workspace


# 例子
```
terraform {
  backend "s3" {
    bucket         = "terraform-aws-consumer"
    key            = "compute/ec2.tfstate"
    region         = "eu-west-1"
    profile        = "ew1.production"
    dynamodb_table = "terraform-consumer"
    encrypt        = "true"
  }

  required_version = ">=1.4.0"
}
```
# 如何创建 

1. 建议登录AWS 管理console, 手动创建， 记得启用version / ACL等功能。

2. 用tf创建，代码如下：
   
```
# Create an S3 Bucket
resource "aws_s3_bucket" "my_bucket" {

  bucket = local.env[terraform.workspace].bucket_name

  tags = merge(try(local.env[terraform.workspace].tags, {}), {
    Name                         = "Your-XXX"
  },var.tags)
}

resource "aws_s3_bucket_versioning" "my_bucket_versioning" {
  bucket = aws_s3_bucket.my_bucket.id
  versioning_configuration {
    status = "Enabled"
  }
}

resource "aws_s3_bucket_public_access_block" "my_bucket_policy" {
  bucket                  = aws_s3_bucket.my_bucket.id
  block_public_acls       = true
  block_public_policy     = true
  ignore_public_acls      = true
  restrict_public_buckets = true
}

resource "aws_s3_bucket_server_side_encryption_configuration" "my_bucket_encryption" {
  bucket = aws_s3_bucket.my_bucket.bucket

  rule {
    apply_server_side_encryption_by_default {
      sse_algorithm = "AES256"
    }
  }

  depends_on = [
    aws_s3_bucket.my_bucket
  ]

}

resource "aws_s3_bucket_lifecycle_configuration" "this" {
  bucket = aws_s3_bucket.my_bucket.id

  rule {
    id = "retention-policy"

    expiration {
      days = 30
    }
    abort_incomplete_multipart_upload {
      days_after_initiation = 30
    }

    status = "Enabled"
  }
}


# Create a separate DynamoDB table for Terraform state locking
resource "aws_dynamodb_table" "terraform_lock" {
  name = local.env[terraform.workspace].dynamodb_name

  billing_mode = "PAY_PER_REQUEST"
  hash_key     = "LockID"
  attribute {
    name = "LockID"
    type = "S"
  }
}
```
